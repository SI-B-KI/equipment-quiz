Jahres-Quiz PWA (installierbar auf iPad/iPhone)

WICHTIG: iOS installiert PWAs nur von einer HTTPS-Website (nicht von file://).

1) Dieses ZIP entpacken.
2) Den gesamten Inhalt (index.html, manifest.webmanifest, service-worker.js, icons/...)
   in einen Ordner auf eine HTTPS-Website hochladen.
   Beispiele: GitHub Pages, Netlify, Azure Static Web Apps, eigener Webspace.
3) Auf dem iPad in Safari die URL zu index.html öffnen.
4) Teilen-Symbol -> 'Zum Home-Bildschirm' (Add to Home Screen).
5) Danach startet es wie eine App (Vollbild).

Hinweis: Service Worker sorgt für Offline-Funktion nach dem ersten Laden.
